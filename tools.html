MC服务器

服务器状态查询

部分服务器休眠时可能显示离线，但游戏内大概率可正常连接。

指令生成器

输入中文，自动生成MC指令。例：给我64个钻石 / 踢了张三 / 封禁李四 / 把Steve传送到0 70 0

管理员常用指令

/gamemode creative
/gamemode survival
/give @p diamond 64
/time set day
/weather clear
/tp @p 坐标
/kick 玩家名
/ban 玩家名
/op 玩家名

地图种子查看器

输入种子号，前往Chunkbase查看地形结构（基岩版专用）

将在新窗口打开Chunkbase种子地图，可查看村庄、要塞等

村民交易查询

选择职业查看常见交易（基岩版）

附魔兼容查询

选择两个附魔，查看能否共存

红石信号计算

输入距离（格），计算红石信号衰减后的强度

自定义计分板生成

填写参数，一键生成计分板创建和显示指令

服务器公告模板

点击即可复制带颜色的公告模板，在游戏里粘贴使用
